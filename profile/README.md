# 🌱 Rhizae Studio

Rhizae Studio is a civic design, data storytelling, and public-interest technology studio.

It explores how accessible interfaces, research tools, and visual systems can make complex public issues easier to understand and act on. Public information is often technically available but practically inaccessible.

Rhizae Studio focuses on the gap between information existing and information being usable. The work here treats design, data, and software as tools for making research, policy, and civic systems more understandable to the people affected by them.

## 🔎 Focus Areas

- ♿ Accessibility-first interface design
- 📊 Data journalism and visual explanation
- 🏘️ Community economic resilience
- 🏛️ Public policy communication
- 🧾 Research transparency
- 🤝 Civic engagement infrastructure

## 🧪 Status

This organization is a working lab. Some repositories are stable projects; others are prototypes, drafts, or research experiments.
